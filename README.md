# Portfolio Fernanda Oliveira

Portfolio estatico preparado para publicacao no GitHub Pages.

## Arquivos principais

- `index.html`
- `styles.css`
- `script.js`

## Publicar no GitHub Pages

1. Crie um repositorio no GitHub.
2. Envie estes arquivos para a branch `main`.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch `main` e a pasta `/root`.
6. Salve e aguarde a publicacao.

## Ajustes recomendados

- Atualizar o link do LinkedIn quando quiser adicionar.
- Substituir ou complementar os textos com metricas finais verificadas.
- Adicionar favicon, imagem social e dominio customizado, se desejar.
